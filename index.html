🔌 إعدادات قاعدة البيانات (Supabase)

اعمل مشروع على supabase.com وحط الـ URL والـ anon key هنا. البيانات بتتحفظ تلقائي في المتصفح فعشان كدا اللي اتسجل لازم تدخله مرة واحدة بس.

🗄️ خطوات إعداد قاعدة البيانات

1) ادخل على مشروعك في Supabase → SQL Editor → نفذ الكود ده مرة واحدة:

2) من Settings → API انسخ الـ Project URL والـ anon public key وحطهم فوق.

3) اضغط "حفظ والاتصال" والصفحة هتبدأ تحفظ كل حاجة في الـ database مباشرة.

🌐 رفع الصفحة على دومين مجاني

الخيارات السهلة:

• Netlify Drop: ادخل على app.netlify.com/drop واسحب الملف ده. خلاص بقيت عندك دومين مجاني.

• Vercel: ارفع على GitHub → اربط مع Vercel.

• GitHub Pages: ارفع الملف في repo وفعّل Pages من الإعدادات.

• Cloudflare Pages: برضه مجاني وسريع.

💾 نسخ احتياطي

تقدر تصدر كل بياناتك كملف JSON وتستوردها مرة تانية لو احتجت.
